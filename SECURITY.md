# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take the security of Aura OS seriously. If you discover a security vulnerability, please do **NOT** open a public issue.

### Process
1.  Email details to **security@auraos.io**.
2.  Include a proof of concept (PoC) if possible.
3.  We will acknowledge your report within 48 hours.
4.  We will provide a timeline for the fix.

### Scope
- Kernel memory safety violations.
- Container escape vulnerabilities (Bottle system).
- Unauthorized access to the Cognitive Kernel / Context Orb.
